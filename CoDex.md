1. При нажатии на логотип происходит переход на главную страницу (страницу подборок) приложения.

   ![genresImage](https://user-images.githubusercontent.com/46110574/192609307-febaaa42-2760-477b-adda-3e98c8f98440.png)
   
4. При нажатии на кнопку `Премьеры` в меню происходит переход на страницу премьер (https://park-akino.ru/premiers), а кнопка окрашивается в розовый цвет.

6. При вводе в строку поиска пустой строки появляется надпись `Запрос некорректен, попробуйте ещё раз`.
   
   ![emptystringsearch](https://user-images.githubusercontent.com/46110574/192609770-1ec4698d-dd08-4206-9db7-a4247bd32232.png)

   ![emptysearch](https://user-images.githubusercontent.com/46110574/192610074-f6985390-70a1-4432-91f5-d391dbf99b6d.png)

    **_БАГ_**: Не появляется надпись `Упс, ничего не найдено` в разделах `Фильмы` и `Персоны`.
    
   ![FindLongString](https://user-images.githubusercontent.com/46110574/192611677-eefb7592-8759-41f2-a6df-e36b407641b4.png)
   
9. При наведении на кнопки в меню, текст кнопок окрашивается с белого на розовый. 

   ![hoveronbutton1](https://user-images.githubusercontent.com/46110574/192610291-531b7bac-919e-404a-b854-86e5f0e42de8.png)

## <a id="movie"></a>**Страница одного фильма**
### Тесты:
1. Если пользователь авторизован, то под кнопкой `Смотреть трейлер` находится кнопка `Добавить в подборку:`.
   
   <img width="294" alt="image" src="https://user-images.githubusercontent.com/46110574/192615564-5c658d9f-2fe4-43a5-ac23-6336bb28eedb.png">
   
2. При нажатии на кнопку `Смотреть трейлер` происходит переход по ссылке на трейлер в сервисе YouTube.
3. Если пользователь авторизован то при нажатии на иконку звёздочки в разделе рейтинг фильма появляется надпись `Ваша оценка: (выбранная оценка). Рейтинг 
фильма: (рейтинг фильма)`
<!-- TODO add screenshot with this text -->
4. Если пользователь не авторизован, то при нажатии на иконку звёздочки в разделе рейтинг фильма появляется надпись `Чтобы поставить оценку, пожалуйста, зарегистрируйтесь`.
5. При нажатии на ссылку `зарегистрируйтесь` в случае отсутствия авторизации происходит переход на страницу авторизации (URL: https://park-akino.ru/login?redirect=/).
6. Если пользователь авторизован, то после раздела отзывы находятся поле ввода отзыва, кнопки `Выберите тип отзыва`, `Оставить отзыв`.
6. Если пользователь не авторизован, то после раздела отзывы находятся находится кнопка `Войти, чтобы оставить отзыв`.
7. Если пользователь авторизован, то после раздела `Отзывы` находятся поле ввода отзыва, кнопки `Выберите тип отзыва`, `Оставить отзыв`.
8. Если пользователь не авторизован, то после раздела `Отзывы` находится кнопка `Войти, чтобы оставить отзыв`.
2. При выставлении оценки фильму, которая равна текущей, актуальная оценка должна повыситься или остаться неизменной; 
2. При выставлении оценки фильму, которая равна текущей, актуальная оценка должна остаться неизменной; 
2. При выставлении оценки фильму, которая равна текущей, актуальная оценка должна остаться неизменной; 
3. При выставлении оценки фильму, которая ниже текущей, актуальная оценка должна остаться неизменной или понизиться.
10. В случае, если пользователь не авторизован, при нажатии на кнопку `Войти, чтобы оставить отзыв`, происходит переход на страницу авторизации.

11. При нажатии на имя актёра в разделе "О фильме" происходит переход на страницу актёра (https://park-akino.ru/actors/`N`, где `N` - это id актёра).

15. При наведении на личную подборку рядом с названием личной подборки появляется иконка галочки/ крестика, соответствующие удалению/добавлению фильма в случае нажатия на название личной подборки.

    ![addToCollection](https://user-images.githubusercontent.com/29610387/191067440-e9f8ce3d-3031-437f-89b1-412a1bb0afbe.png)
   
16. При наведении на личную подборку рядом с названием личной подборки появляется иконка галочки, соответствующая добавлению фильма в случае нажатия на название личной подборки.
17. При наведении на личную подборку рядом с названием личной подборки появляется иконка крестика, соответствующая удалению фильма в случае нажатия на название личной подборки.


18. При нажатии на имя пользователя в списке отзывов происходит переход в профиль соответствующего пользователя.
19. При нажатии на кнопку `Выберите тип отзыва` появляется dropdowm меню с возможностью выбрать один из трёх типов отзыва отлично/неплохо/ужасно, которые соответствуют цветам зеленый/серый/красный.
20. Максимальный размер поля ввода содержимого отзыва равен 300 символам. 
   
   ![maxCommentInput](https://user-images.githubusercontent.com/46110574/192617813-f83e4084-1598-4ffd-8cdf-1b63f395871c.png)
   
21. Минимальный размер поля ввода содержимого отзыва равен 15 символам.

    **_БАГ_**: Минимальный размер поля ввода содержимого отзыва равен 0 символов.
   ![EmptyCard](https://user-images.githubusercontent.com/29610387/191066742-135cc144-1c94-444f-9195-d119cc6cd6e7.png)
22. Ввод содержимого отзыва средней длины (>15 && <300 символов), отзыв оставляется:
    - с использованием латинских букв
    - с использованием цифр
    - с использованием знаков препинания
                                                       
    **_БАГ_**: Отсутствие переноса строки, если она больше 100 символов (при масштабе 100 процентов) и не имеет пробельных символов.![OverCard](https://user-images.githubusercontent.com/29610387/191066753-e2899281-d0bd-459b-8b70-62544852fb26.png)
